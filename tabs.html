settings

Option
Type
Values (default)
Description
defaultTab
number
0
Selects the tab to activate when page first loads.
animation
string
"show", "fade", "slide" ("show")
Animation tab change.
speed
number
300
Animation speed change tabs.
direction
string
"left", "top", "right", "bottom" ("left")
Direction animation. Works only if you selected animation: "slide"
scope
string
"200px"
Scope animation. Works only if you selected animation: "slide"
autoplay
boolean
false
Autoplay tab change.
interval
number
5000
Interval change tabs on autoplay.
hash
boolean
false
Tells easyTabs whether or not to update the browser window's URL hash, useful for SEO and bookmarking.
beforeChange
function
A function that is played before animation
afterChange
function
A function that is played after animation
